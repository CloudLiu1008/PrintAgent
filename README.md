# ERP PrintAgent

ERP PrintAgent is a Windows local printing agent for jewelry ERP label printing. It runs as an Electron tray app and exposes a localhost HTTP API for the ERP web app.

## Features

- Local HTTP service on `http://127.0.0.1:17688`
- Health check, printer list, template label rendering, label printing and test label printing
- Barcode and QR code rendering from ERP label template JSON
- Silent printing through Electron `webContents.print`
- Tray menu with status, settings, test print, logs and exit
- Local config and logs under `%APPDATA%/ERP-PrintAgent`
- Token header protection with `X-PrintAgent-Token`

## Install Dependencies

```bash
npm install
```

## Run

```bash
npm run dev
```

After startup, open:

- `http://127.0.0.1:17688/` for the local status page
- `http://127.0.0.1:17688/health` for the public health check
- `http://127.0.0.1:17688/printers?token=local-secret-token` for browser debugging

Protected API endpoints return `{"success":false,"message":"Unauthorized"}` when the token is missing. ERP calls should send:

```http
X-PrintAgent-Token: local-secret-token
```

## Build Installers

Installers are generated by `electron-builder` into `tools/print-agent/release/`.
Application icons are stored under `build/icon.png`, `build/icon.ico`, `build/icon.icns`, and the runtime tray icon is stored at `src/assets/icon.png`.
Run `npm install` before packaging. The build uses the local Electron runtime under `node_modules/electron/dist`, caches builder tools under `.runtime/electron-builder-cache`, and defaults electron-builder binary downloads to `https://npmmirror.com/mirrors/electron-builder-binaries/`, so it can package more reliably when GitHub downloads are slow or unavailable after dependencies are installed.

The build script sets `HTTP_PROXY` and `HTTPS_PROXY` to `http://127.0.0.1:7897` when no proxy environment variables are already configured. Override these environment variables before running the build if another machine uses a different proxy port.

### Windows

Run on Windows:

```bash
npm run dist:win
```

Expected output:

- `release/ERP-PrintAgent-Setup-<version>-win-x64.exe`

The Windows installer uses NSIS, creates Start Menu and desktop shortcuts, and keeps user config under `%APPDATA%/ERP-PrintAgent` across upgrades.

### macOS

Run on macOS:

```bash
npm run dist:mac
```

Expected output:

- `release/ERP-PrintAgent-<version>-mac-x64.dmg`
- `release/ERP-PrintAgent-<version>-mac-arm64.dmg`
- matching `.zip` archives

macOS packages are unsigned in the first version. On a fresh Mac, the user may need to allow the app in System Settings > Privacy & Security. For public distribution, add an Apple Developer ID certificate and notarization later.

### Development Package

To create unpacked app folders without installers:

```bash
npm run pack:win
npm run pack:mac
```

Build macOS installers on macOS. Windows can build the Windows installer; cross-building macOS installers from Windows is not reliable because DMG signing/notarization tooling depends on macOS.

## API

### GET /health

No token required.

### GET /printers

Requires `X-PrintAgent-Token`.

### POST /print/html

Requires `X-PrintAgent-Token`.

```json
{
  "printerName": "HPRT D35BT",
  "silent": true,
  "copies": 1,
  "widthMm": 40,
  "heightMm": 30,
  "html": "<html>...</html>"
}
```

### POST /render/label

Requires `X-PrintAgent-Token`. Renders ERP template JSON and rows into printable HTML for preview.

```json
{
  "template": {
    "templateName": "SKU标签",
    "widthMm": 50,
    "heightMm": 30,
    "templateJson": "{\"widthMm\":50,\"heightMm\":30,\"elements\":[]}"
  },
  "rows": [
    { "skuNo": "es00001-01", "productCode": "es00001" }
  ]
}
```

### POST /print/label

Requires `X-PrintAgent-Token`. Renders ERP template JSON and rows, then sends the generated HTML to the selected printer.

```json
{
  "printerName": "HPRT D35BT",
  "copies": 1,
  "template": {
    "templateName": "SKU标签",
    "widthMm": 50,
    "heightMm": 30,
    "templateJson": "{\"widthMm\":50,\"heightMm\":30,\"elements\":[]}"
  },
  "rows": [
    { "skuNo": "es00001-01", "productCode": "es00001" }
  ]
}
```

### POST /print/test

Requires `X-PrintAgent-Token`.

## Windows Printer Setup

For label printers such as HPRT D35BT, set the same label size in all three places:

1. ERP template size
2. HTML `@page size`
3. Windows printer preferences paper size
